Numerical Values

Using Numerical Values is straightforward. Some properties expect an integer-in other words a whole number. Others accept either a whole number or a decimal value. Numbers must not contain spaces or the thousands seperator. Decimal fractions use a decimal point (.). You cant use a comma as the decimal point, as is common use in some Europian Countries.

Legth

The CSS specifications use the term lenght to refer to vertival and horizontal measurements.The CSS3 Values and Units module introduces several new units of measurements, as well as a function that allows the browser to calculate a lenght value.

Shrinking Em's

Using em to specifiy font size can result in even shrinking text because nested elements inheret the size from thier parent, resulting in on multiplier effect. For example, if you set the font size of unordered lists to 0.75em (the equivalent of 12px), the text in a nested lists is 0.75 times the size of its parent -- in other words, 9px. At the next level, the text shrinks even further.

    Physical and CSS Pixel

    CSS Pixel. At a distance of 28 inches (71cm), a reference pixel (or CSS Pixel) is equivalent to 1/96 inches (0.26 mm). When displayed on a device that is intented to be viewed to a greater displat--a projection, for example, the device is expected to scale up pixel measurement proportionaly. The long and short of it is that using pixel as the unit of measurement should result in roughly the same size, regardless of the physical pixel density of the screen.

    Using the calc() Function to Compute Length Values

    The situation frequently arises in fluid or flexible layouts where you want to combine absolute values with relative ones. For example, you might want an element to fill 70 percent of the horizontal space, but to have a 10-pixel margin on either side. Because you have no idea how wide the browser viewport will be, its impossible to know how much horizontal space remains alongside. The CSS3 calc() function is designed to solve such problems.

    The calc() function supports the following arithmethic:

    +(add)
    -(subtract)
    /(divide)
    *(multiply)

    mod(modulo division)br

    To calculate the width available alongside the element in the preceding example, CSS3 allows you to specify the value like this:

    cal(30% - 20%)

    Percentage

    Values specified as percentages use a number immediately followed by the percentage sign (%). There must be no space between the number and %. Percentages are always relative to another value, normally the size of the parent element. Browsers are capable of handling percentages to several decimal places.

    Angles

    Units Meaning
    deg Degrees
    grad Gradians (1/400 turns
    rad Radians
    turn Turns (360 degrees)

    Angles are used only with CSS3 properties, such as transform. As with all other units of measurement, there must be no space between the number and unit. Browsers convert angles to the range 0–360°. For example -90deg is the same as 270deg.

    Time

    Unit Meaning
    s Seconds
    ms Milliseconds (1/1000 of a second)

    Time units are used only with CSS3 properties. There are just two units. Again, there must be no space between the number and the unit.

    Using Hexadecimal Notaions

    The most common way of representing color in web design is to use the Hexadecimal Notation for red, gree and blue (RGB). Using hexadecimals numbers allows you to select 256 colors of each values, representing 16 Millions colors. Most graphics and HTML editing programs generate the hexadecimal numbers for you automatically when you use color picker or eyedropper tool. You can use online utilities such as www.colorpicker.com

    When using hexadecimal notation, Remember the following:

    • The color value must begin with the hash or pound sign (#). Omitting the # is a common cause of errors.
    • You can use six-digit or three-digit hexadecimal values (see the sidebar “Shorthand Hexadecimal Values” for an explanation of the difference).
    • Dont mix up the letter O with 0 (zero) or lowercase L with the number 1. The only letters permitted in hexadecimal values are A-F.
    • The letters A-F in a hexadecimal value are case-insensitive. It doesnt matter whether you use uppercase or lowercase, or even a mixture of the two.

    SHORTHAND HEXADECIMAL VALUES

    Hexadecimal color values are normally expressed using six digits. The first pair represents the red component at the color; the second pair represents the green component, and the final pair the blue component.
    Example:

    #redred,greengree,blueblue;

    Using RGB Format

    You can us the rbg() function in style sheet. You specificaly use red, gree and blue values as a comma-seperated list inside the parenthises. The rgb() function accepts the values either numbers in range 0-225 or as percetage.

    rgb(225,0,0)
    rbg(100%,0%,0%)

    Using HSL Format

    To specify colors using hue, saturation and lighting. Hue is represented by an angle on the color circle, with red at the top(0 degrees), green one third at the way around the circle in clockwise direction(120 degrees), and blue two-thirds at the way around(240 degrees). Saturation represents the intensity of the color, making duller or brighter. And lightness makes the color darker or light.

    HSL Format

    Using hsl() function, which takes three comma-seperated values, as follows:

    • Hue expressed as number between 0 and 360: This is the angle of the color on the color circle. Browsers are expected to adjust out-of-range numbers, so-120 should be interpreted as 240(blue).
    • Saturation expressed as percentage: 0% is a shade of gray, and 100% is full saturation.
    • Lightness expressed as a percentage: 0% is black, 50% is normal and 100% is white.

    The follwing example represents red:
    hsl(0, 100%, 50%)

    Creating Transparent Colors with RGBA and HSLA Formats

    RGB and HSL formats allows you to set the opacity of the color using alpha transparency. To specify a color with ampha transparency, use the rgba() or hsl() function. Both funtion take four comma-seperated values. The first three values are the same as rbg() and hsl(). The four values sets the level of transparency, and must be a number between 0(completely transparent) and 1(completely opaque). Setting the alpha of a color not only reduce the color intensity, but it also allows any background to show through. The background becomes more visible the closer the fourth value is to 0.

    The following examples creates semitransparent red:
    rbg(225, 0, 0, 0.5)
    rbga(100% ,0% ,0%, 0.5)
    hsla(0, 100%, 50%, 0.5)
