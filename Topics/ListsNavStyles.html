Styling Lists and Navigation Menus

Styling Unordered and Ordered Lists

CSS has three individual properties and a shorthand property designed to style bullted and numbered lists. They are listed below.

List Properties in CSS
Property Initial Value Description
list-style-type disc Determines the type of symbol used. By default, unordered lists use a solid disc, and ordered lists use a number followed by a period
list-style-position outside Controls whether the symbol or number is displayed as a hanging indent(default) or nested inside the list item
list-style-image none Allows you to use your own image in place of an automatically generated symbol
list-style See individual properties Shorthand property that lets you specify all properties in a single declaration
All properties above are inherited, so the same values are used by nested list unless you create specific rules to override them.

Changing the Position of the Symbol

The list-style-position property accepts just two vakues, namely:

  • inside This tucks the bullet or number inside the list item, so that it is flush with the left edge in left-to-right languages like English
  • outside This is the default position, which puts the bullet or number outside the list item like hangging indent
The styles in list-style-position include the following ckass:

  • By default, the bullet or number is outside the list item
  • You can change this by setting list-style-position to inside
  1. Order lists are treated exactly the same
  2. The text is indented, but the number remains outside

Replacing the Symbol with Your Own Image

The list-style-image property allows you to replace the symbol with an image of your own. Although it works with both ordered and unordered list, the same image is used for every item. So, in practical terms, it makes sense only with unordered lists.
As shown below, the image used in place of the default disc. The image used in this example is 30x30px, which produces a larger shape than the defualt bullet. As a general rule, the height of the image shouldnt exceed the font size of the text.

Using the list-style Shorhand Property

The list-style shorthand property accepts the same values as the individual list-style-type,list-style-position, and list-type-image properties. You can define all three values, seperated by a space, or just one or two. Any value that is omiited uses the initial value listed before.

ul{ list-style-image: url(../yawa/animal.png); list-style-type: square; }}
ul{ list-style: url(../yawa/animal.jpeg) square; }
