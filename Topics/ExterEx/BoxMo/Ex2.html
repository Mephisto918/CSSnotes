Examining the 2nd exameple, these can be seen:

  • The background and border of the inline <span> obscures the preceding line
  • The next line displayed over the background and border of the inline <span>
  • The text in the inline <span> wraps normally, splitting the box over multiple lines
  • If you resize the brownser window, the inline block is never split.

In other words, setting the display property to inline-block gives the element all the characteristics of a block-level element, but displays it inline. You can verify this in 2nd display which adds text-align and vertical align properties to the style rules.
