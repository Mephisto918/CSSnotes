Creating Gradients Without Images

CSS Gradient Basics

Although CSS gradients dont use images, you apply them to an element using background-imgge property or the shorthand background property. You define a gradient by setting the property value with one of the four functions listed below.

CSS Gradient Functions
Function Description
linear-gradient() Sets the direction and colors of a gradient that fades from one color to the next in a traight line. The direction can be set using keyword or an angle. If the direction is omitted, the gradient is drawn from the top to bottom.
radial-gradient() Creates a circular or elliptical gradient, and optionally sets its shape, size, and position. If all optional features are ommited, an elliptical gradient is centered in the elements background.
repeating-linear-gradient() Creates a linear gradient, in which the colors are repeated infinitely in both directions
repeating-radial-gradient() Creates a radial gradients that represents infinitely

Browsers draw the gradient inside a gradient box. By default, this fills the elements padding box. The gradient does not have any intrinsict dimentions, but you can control its size and position in the same way as a background image using background-size and background-posiiton properties.

Setting the Position of Color Stops

The position of a color stop along the gradient line or ray can be specified using a length or percentage. Lengths are measured from the starting point. Percentages are relative to the distance between the starting and ending points. Creating Linear Gradients

You can control the direction of the gradient using keywords to point to one of the sides or corners, which then becomes the ending point. Alternatively, you can specify the angle at which the gradient should be drawn

To create a linear gradient, you use the linear-gradient() fucntion, which takes the following comma-seperated arguments:

  • The angle or direction of the gradient(optional)
  • A comma-seperated list of two or more color stops

Using Keywords to Set the Direction of the Gradient Line

The linear-gradient() function supprts the following keywords to set the ending point of the gradient line: top, bottom, right and left. On their own, they represent the side of the gradient box; but they can also be combined to indicate a corner, such as top left. To set the direction, you precede the keywords with to.

#direction{ width: 200px; height: 100px; background-image: linear-gradient( to top left, blue, red); }

Setting the Angle of the Gradient Line

You can use an angle instead of keywords to set the direction of the gradient line. Specifying the angle in degrees with the deg unit of the measurement is simple, but there are two complication factors:

  • In all browsers, most angles produce starting and ending points outside the gradient box.
  • Browsers that require a browser-specific prefix with linear-gradient() use a different coordinate system from the final specification

Creating Radial Gradients

A radial gradient emerges from a single point with the color spreading outward in a circular or elliptical shape

You define the gradient with the radial-gradient() function, which optionally sets the size and position of the circle or ellipse. The browser-specific functions use a similar syntax to the standard one, but there are some significant differences.

Standard Syntax

The radial-gradient() function takes the following arguments:

  • Shape The accepted value are circle and ellipse
  • Size: This determines the extent of the radial gradient, either using specific dimensions or keywords
  • Position: This determines the center of the circle or ellipse. The value is expressed in the same way as background-position, preceed by the keyword at. If ommited, it defaults to at center.
  • Color stops: A comma-seperated list of colors with optional positions along the gradient ray starting at the center
The only required argument is the list of color stops. All other arguments are optional

Setting the Shape of the Gradient

You can set the shape of the gradient explicitly by using keyword circle or ellipse. If you omit the shape, it defaults to circle if the size is a single length. In all other cases, it defaults to ellipse.

Setting the Size of the Gradient

You can set the size of the circle or ellipse explicitly by using keywords

  • A circular gradient accepts only a single length, which sets its radius
  • An ellipse gradient accepts two values, which can be either lengths or percentages. The first value sets the ellipse horizontal radius. The second one sets its vertical radius. Percentage values are relative to the width and height of the gradient box, respectively.
Both shapes accepts any of the following keywords to determin thier size:
  • closest-side
  • farthest-side
  • closest-corner
  • farthest-corner

Setting the Position of the Gradient

You can position the gradient using the at keyword followed by an expression that is valid for the background-position property. For example, the circular gradient is positioned 100px from the left and 50px from the top.

background-image: radial-gradient(circle closest-side at 50% 50%, blue, red, green);

Creating Repeating Gradients

The functions for repeating gradients, repeating-linear-gradient() and repeating-radial-gradient(), take the same arguments as their none repeating equivalents. However, its portant to position the last color stop before the ending point. If you dont, it defaults to 100%, preventing the patterns from being repeated.

Creating Diagonal Stripes

A simple, yet effective use of a repeating linear gradient is to create diagonal stripes. This sets the angle of the gradient line to -45deg, and creates two color nonestop, the first one is dark blue. No position is declared, so it defaults to 0%. The second color stop is white, and its position at 6px along the gradient line. So, the gradient transitions from dark blue to white in a very short space. The function then repeats the same pattern infinitely, producing the result below.

#repeat{ width: 200px; height: 100px; background-image: repeating-linear-gradient(-45deg, #426A77, #FFF 6px) }

Creating a Smooth Repeating Gradient

The repeating gradient in the preceing example creates the illusion of stripes because there are only two color stops and the distance between them is very short. To create a more fluid repeating gradient, the first and last color stops need to be the same color.

The dark yellow is used as the first color stop. It has no declared position, so it defaults to the starting point of the gradient line. The lighter yellow is the second color stop at 5% and the first color is repeated at 10%. This produces the ridge effect shown below

#repeat-smooth{ width: 200px; height: 100px; background-image: repeating-linear-gradient(to right, #D9CC3C, #FFEB79 5%, #D9CC3C 10%); }

Using Multiple Gradients

Because CSS gradients are applied using the background-image property, you can add multiple gradients as the background to an element by listing definitions seperated by commas. The gradients are applied in revers order, with first one appearing in front. Because gradients normally fill the entire background, you need to set the final colorstop to transparent on gradients that appear in front. Alternatively, you can control the size and position of individual gradients with the background-size and background-posiiton position.

Radial gradients using the standard syntax are particulary suited for use as multiple gradients because the radial-gradient() function allows you to set the size on the gradient precisely using lengths (for both circle and ellipse) or percentage (for ellipse).

The first radial gradient is a 12px circle positioned 25px from the left and 30px from the top. Its first color stop is white with 80% alpha transparency, and its final color stop is transparent.
The second gradient appears behind the first one. Its also a circle, positioned at 30% both horizontally and vertically, offsetting it to the upper left. No size has been declared,so it fills the background to farthest-corner. The three color stops are increasingly deepr shade of red
