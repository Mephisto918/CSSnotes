Changing the Look of the Cursor

General Purpose Cursor

Keyword Example Description
auto cursor Browser selects cursor base on current context.
default cursor Default cursor - usually an arrow
none cursor Cursor is hidden

Links and Status Cursors

Keyword Example Description
context-menu cursor Context menu is available for the current object
help cursor Help is available
pointer cursor Indicates a link
progress cursor Indicates something is happening, but that user still interact with the browser
wait cursor Indicates the browser is busy and user should wait

Selection Cursor

Keyword Example Description
cell cursor Indicates a cell or set of cells may be selected
crosshair cursor Display two thin interacting lines
text cursor Indicates text can be selected
vertical-text cursor Indicates vertical-text can be selected

Drag and Drop Cursors

Keyword Example Description
alias cursor Indicates that an alias or shortcut is to be created
copy cursor Indicates that something is to be copied
move cursor Indicates that somethingis to be moved
no-drop cursor Indicates tbat the dragged item cannot be droppped at the current location
not-allowed cursor The requested action will not be carried out
